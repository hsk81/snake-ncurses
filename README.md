# Snake

Keep eating the yummy pound signs till you bite yourself! ;)

## Build

make

## Run (release)

./dist/Release/GNU-Linux/snake

## Run (debug)

./dist/Debug/GNU-Linux/snake

## Quit

CTRL+C

## Controls

Use up, down, left and right arrow buttons.
