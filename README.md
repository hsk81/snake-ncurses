# Snake

Keep eating the yummy pound signs till you bite yourself! ;)

## Build
```bash
$ make
```

## Run
```bash
$ ./dist/Release/GNU-Linux/snake
```

## Quit
```
CTRL+C
```

## Controls

Use `up`, `down`, `left` and `right` arrow buttons to control the snake.
